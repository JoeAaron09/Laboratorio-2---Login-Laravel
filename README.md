# Laboratorio-2---Login-Laravel

Prerrequisitos:

- PHP versión 8.0 o superior

- Composer versión 2.9.5

- Laravel installer

- Paquete de WampServer

- Servidor Web: Apache

- Base de Datos: MySQL

- Editor de código: Visual Studio Code

- NPM versión 11.12.1

- Sistema Operativo: Windows
